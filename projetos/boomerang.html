BOOMERANG

Local: Rio de Janeiro. Sistema: GESTÃO DE SERVIÇOS Sistema que cuida de todo o ciclo de serviços da empresa, desde a proposta ao cliente, passando pelo contrato e gerindo todo o pessoal selecionado para os serviços contratados. Com base em dados cadastrados, o sistema seleciona as pessoas para o "job" mostrando informações específicas dos cargos/pessoas necessárias para atender cada contrato específico. O sistema permite, ainda, tratar todo o fluxo financeiro referente ao contrato.
